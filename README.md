# Google Apps Script

https://www.google.com/script/start/

### プロジェクトを作成する
1. https://www.google.com/script/start/ にアクセス
2. 画面右上の「Start Scripting」をクリックする
3. 画面左上「新しいプロジェクト」をクリックする
